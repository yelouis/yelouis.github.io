# Louis Ye | Software Engineer Portfolio

A custom 3D flip-card interactive portfolio website built entirely with vanilla HTML5, CSS3, and Anime.js.

## Features
- **3D Card Deck**: A completely custom interaction system that simulates a deck of playing cards in a 3D perspective space.
- **Dynamic Navigation**: Supports horizontal swiping (pointer events), UI buttons, and Arrow Keys to flip through the deck.
- **Procedural Theming**: Fully automatic Snorlax-inspired color palette randomizer for each instantiated card. 

## Credits
- **Animation Engine**: Anime.js (animejs.com)
- **Background Assets**: Snorlax SVG (PokeAPI)
